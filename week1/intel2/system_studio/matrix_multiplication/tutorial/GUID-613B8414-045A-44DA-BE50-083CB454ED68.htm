Tutorial: Using Intel® System Studio Components for Matrix Multiplication Computation

Building the Sample with Eclipse* IDE on the Host and Executing it on a Target

In this sample, Yocto Project* is used as the target operating system. Refer to Intel® System Studio System Requirements to know more about supported operating systems.

Step-1. On the Host Linux system, run the following command: 

> source /opt/poky/1.7/environment-setup-i586-poky-linux
> ./eclipse

Another option is to configure the script in ~/.bashrc (or /etc/profile) files, this will set up the Yocto toolchain in environment variables.

Step-2. Create project with existing source code and Intel® System Studio as the Toolchains as described in before.

Step-3. Configure it to build for Yocto target:

Click Project->Properties and find C/C++ Build->Settings, and configure as shown below:

Step-4 . Configure the toolchain (example: 32bit or 64bit targets) in the Intel® C++ Compiler tab as shown below, select the [x86] toolchain and click Use Selected as build it for Yocto 32bit, you may use [x86_64] if you are targeting for 64bit Yocto:

Step-5. Click Project->Build Project to build the project and generate the output in the Console window:

Step-6. To run this sample code on the target system, copy the generated binary onto the target with ssh/scp.

Note

You can also set up SSH/SCP in Eclipse to run on remote target, explore it by yourself from Eclipse documents.
