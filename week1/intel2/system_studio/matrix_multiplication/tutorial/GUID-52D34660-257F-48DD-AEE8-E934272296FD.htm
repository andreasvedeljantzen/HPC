Using Intel® VTune™ Amplifier for Systems to Identify Performance Bottlenecks

Intel® VTune™ Amplifier for Systems is a performance analysis tool for users developing serial and multithreaded applications on embedded and mobile systems. It provides a rich set of performance insight into hotspots, threading, locks & waits, OpenCL™ kernels execution, bandwidth and more.

In this part of the sample code, you will analyze the performance bottleneck in the matrix multiplication code.

  1. In the Eclipse* menu, select Intel System Studio - > Intel VTune™ Amplifier for Systems -> Intel VTune Amplifier for Systems to open the VTune Amplifier GUI:
  2. Select New Project in the VTune Amplifier menu and specify the matrix multiplication sample application as a local analysis target:
  3. Switch to the New Analysis tab, select an analysis type (for example, Basic Hotspots) and click Start to run the the analysis:

Refer to the VTune Amplifier user guide for details on configuring the analysis: https://software.intel.com/en-us/node/544018

Analyze the collected result. In the example below, the CPU Time hotspot is detected in the inner for loop of the matrix multiplication code. By parallelizing this part of source code, we can try to reduce the CPU Time.

To understand the complete analysis result of the VTune Amplifier, refer to the user guide: https://software.intel.com/en-us/node/609461
