Using Optimized Functions From Intel® Math Kernel Library

The Intel® Math Kernel Library (Intel® MKL) offers accelerated math processing routines that improve application performance and reduce development time. Intel® MKL includes highly vectorized and threaded matrix manipulation routines, fast Fourier transform (FFT) routines, and vector math and statistical functions. For more information, see https://software.intel.com/en-us/intel-mkl

The Intel® MKL sample computes the real matrix product C=alpha*A*B+beta*C using a triple nested loop, where A, B, and C are matrices and alpha and beta are double precision scalars.

1. Set 03_mkl_matrix_multiply as Startup project in Microsoft* Visual Studio*.

In the matrix multiplication example, replace each of the for() loops with a call to the Intel® MKL function cblas_dgemm() and pass the appropriate parameters. Refer to the Intel® MKL developer reference for more information about the functionality at https://software.intel.com/en-us/intel-mkl-support/documentation.

Select Use threaded MKL library (-mkl=Parallel):

Build ISS_03_‘mkl_matrix_multiply

Double click C/C++ Application to create the configuration with specific project name:

Click the Environment tab and add new Value for Intel MKL. Set LD_LIBRARY_PATH as <install_dir>/compilers_and_libraries_2017/linux/mkl/lib/<ia32/intel64>:

Build and run the example and note the performance number.
