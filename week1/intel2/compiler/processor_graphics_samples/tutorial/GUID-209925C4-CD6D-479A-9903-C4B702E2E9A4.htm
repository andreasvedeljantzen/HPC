Preparing the Sample Application

Follow these steps to begin:

  1. Start Microsoft* Visual Studio and open the file processor_graphics_solution_<year>.sln (where <year> refers to the version of Microsoft* Visual Studio you are using).

  2. In Solution Explorer, right click the convolution project and then click Intel C++ Compiler > Use Intel C++.

In the next step, you will build the source code into an application that runs on both the CPU and coprocessor.
