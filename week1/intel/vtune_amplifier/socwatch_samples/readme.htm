The SoCWatch sample is a simple numerical computation of Pi designed to demonstrate the impact of CPU operations on the power and energy of any Windows* system. Use this program with Intel® Energy Profiler tutorials to better understand and improve power management on modern systems.

System Requirements

  • Hardware: Refer to the Intel® SoC Watch for Windows* OS User's Guide for hardware requirements
  • Software:
    • Microsoft Windows* OS
    • Microsoft Visual Studio 2012* standard edition or later. Note: Depending on your version of Visual Studio, you may need to install the MFC library for Visual Studio.

Build Instructions

A pre-built binary already exists in the socwatch_samples/bin directory. The pre-built binary is be used in the tutorial.

Use the following steps to create a new build:

  • Using Microsoft Visual Studio 2010 or 2012:
    1. Open the socwatch_samples.sln file.
    2. Choose the Debug configuration option from the Configuration Manager.

    3. Select the Build > Build Solution action.

      The new binary is available from the bin directory after the build succeeds.

  • Using a command prompt:
    1. Enable your Microsoft Visual Studio compiler environment.
    2. Open the Microsoft Visual Studio command prompt.
    3. Navigate to the socwatch_samples folder.
    4. Compile the binary using the following command: Build.bat
    5. Clean the build using the following command: Build.bat clean
