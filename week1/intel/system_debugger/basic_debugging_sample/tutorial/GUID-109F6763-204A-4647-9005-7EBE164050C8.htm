Overview

About This Tutorial

This tutorial uses the sample sample_os.bin and guides you through the basic steps of debugging an application with the Intel® System Debugger.

Estimated Duration

20 minutes

Learning Objectives

After completing the tutorial, you should be able to:

  • launch the debugger with a preconfigured script
  • set a breakpoint at a specific line of code
  • set a breatkpoint at a specific function
  • evaluate local variables of a function
  • display and view debugging information, such as assembly code, memory, descriptor tables, address translation, function callstack, system and processor specific registers
