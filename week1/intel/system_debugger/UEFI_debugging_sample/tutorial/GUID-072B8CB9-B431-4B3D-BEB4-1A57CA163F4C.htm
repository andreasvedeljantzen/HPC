Overview

The Intel® System Debugger supports UEFI debugging via the UEFI agent, a built-in debug module which runs on the target providing basic debug capabilities over serial, USB or TCP channel.

About This Tutorial

This is a set of tutorials that introduce UEFI debugging features of the Intel® System Debugger using an emulator for the first two tutorials and a real hardware board for the third one. All three tutorials use the UEFI agent connection to the target.

Estimated Duration

10 minutes per tutorial

Learning Objectives

After completing all tutorials, you should be able to:

  • run a pre-built binary in an emulator and perform basic UEFI debugging, such as displaying the DXE system table and modules
  • compile your own code and run it in an emulator
  • set up the firmware build tree for a development board, enable the UEFI debug agent, build and flash the firmware and connect to the board over a serial connection
