Using Intel® Cilk™ Plus to Parallelize the Application

Intel® Cilk™ Plus is an extension to C and C++ that offers a quick and easy way to harness the power of both multicore and vector processing. Intel Cilk Plus provides a simple yet surprisingly powerful model for parallel programming, while runtime and template libraries offer a well-tuned environment for building parallel applications. Find more information at https://software.intel.com/en-us/intel-cilk-plus.
