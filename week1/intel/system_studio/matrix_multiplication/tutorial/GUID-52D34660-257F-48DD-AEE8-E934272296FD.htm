Using Intel® VTune™ Amplifier for Systems to Identify Performance Bottlenecks

Intel® VTune™ Amplifier for Systems is a performance analysis tool for users developing serial and multithreaded applications on embedded and mobile systems. It provides a rich set of performance insight into hotspots, threading, locks & waits, OpenCL™ kernels execution, bandwidth and more.

In this part of the sample code, you will analyze the performance bottleneck in the matrix multiplication code.

  1. Click the Intel® VTune™ Amplifier for Systems icon on the toolbar or select Intel VTune Amplifier version for Systems > New Analysis from the Tools menu:



  2. Select and run the Advanced Hotspots analysis in the Algorithmic Analysis category:



Refer to the VTune Amplifier for Systems user guide for additional configuration information.

As shown in the picture below, the hotspot is in the inner loop of the matrix multiplication. By parallelizing this part of source code, we can try to reduce the CPU Time.
