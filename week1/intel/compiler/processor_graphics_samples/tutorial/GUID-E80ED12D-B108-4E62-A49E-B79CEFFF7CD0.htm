Preparing the Sample Application

In this tutorial, you will use the following files:

  • Makefile, which you use to build, run, and report the result of the samples.

  • readme.html, which describes the included samples.

  • The source code in src/ under each sample folder which supports each sample.

In the next step, you will build the source code into an application that runs on both the host and the target.
