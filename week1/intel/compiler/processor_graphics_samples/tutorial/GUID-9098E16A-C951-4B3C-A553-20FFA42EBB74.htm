Building the Source Code

In this step, you will compile the source code into an application that runs on both the host and the target. After you compile and run the application and then examine the output.

  1. Click Debug > Start Without Debugging.

  2. In the Microsoft Visual Studio dialog box, click the Yes button.

    When compilation completes, a command window appears. Find the following statement at the beginning output:

    This program is built with __INTEL_OFFLOAD

    The statement indicates that the sample code was compiled into an application that runs both on the host and the target.

    Note of the best time per iteration, which is displayed at the end of the output. You will compare this time to the best time per iteration of application built to run on just the host.

  3. Press any key to close the command window.

In the next step, you will look at the source code to learn how to define which code section should run on the target and how the offload session is organized.
